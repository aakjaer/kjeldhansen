# README

Public website for Connected Cars A/S

# Installation

## MacOSX

```
brew install ruby
gem install bundle
bundle install
gem install jekyll
```

## Ubuntu

```
apt-get install ruby ruby-dev build-essential
gem install bundle
bundle install
gem install jekyll
```

## Windows

```
choco install ruby -y # Restart the command prompt to get ruby and gem in path
gem install bundle
bundle install
gem install jekyll
```

# Preview site changes

```
jekyll serve
```
